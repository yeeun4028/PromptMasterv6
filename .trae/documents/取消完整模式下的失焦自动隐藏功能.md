我将修改 `MainWindow.xaml.cs` 中的 `Window_Deactivated` 事件处理程序，以实现“完整界面失去焦点不自动隐藏，而迷你窗口保持自动隐藏”的需求。

**具体变更计划：**

1. **文件**：`MainWindow.xaml.cs`
2. **方法**：`Window_Deactivated`
3. **修改逻辑**：

   * 在执行自动隐藏逻辑（启动 `_hideTimer`）之前，增加一个判断条件。

   * 检查 `ViewModel.IsFullMode` 是否为 `true`。

   * 如果是完整模式，则直接 `return`，不再启动隐藏定时器。

   * 保留 `this.Topmost = false` 的逻辑，确保完整窗口失去焦点后不再置顶，符合常规窗口行为。

**预期效果：**

* **完整模式**：失去焦点时，窗口不再消失，但会取消置顶（允许被其他窗口覆盖）。

* **迷你模式**：保持原有行为，失去焦点后延迟自动隐藏。

代码修改位置示意：

```csharp
// [MainWindow.xaml.cs]

private void Window_Deactivated(object sender, EventArgs e)
{
    // ... (保留现有代码：Topmost = false)

    StopHideTimer();

    // ★★★ 新增代码 ★★★
    // 如果是完整模式，不执行自动隐藏
    if (ViewModel != null && ViewModel.IsFullMode)
    {
        return;
    }

    // ... (保留现有代码：启动 _hideTimer)
}
```
